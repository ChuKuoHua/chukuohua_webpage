  • HTML + CSS

  • 最熟悉的程式語言，了解 HTML、CSS 基本架構，能套用 transform、transitio 製作動畫，活用 position 定位物件．

  • 現今已 Sass 編輯樣式，因為有 CSS 觀念很快就掌握撰寫訣竅．使用 Sass 規劃 Web layout 及做出響應式網頁，未來將會更深入的探討及研究．

  • javaScript ＆ jQuery

  • 在學時期雖有撰寫過，但接觸的不多，只能透過線上課程重新學習基礎與運用．了解的語法只是冰山一角，未來會不停磨練，並透過網路上的資源提升自己的撰寫能力．

  • PHP&MySQL

  • 在學時期，專題製作所接觸的程式語言，接觸的不深入，因此只了解一些基本觀念．
